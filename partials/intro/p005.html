JQuery UI Selectable

Enable a DOM element (or group of elements) to be selectable. Draw a box with your cursor to select items. Hold down the Ctrl key to make multiple non-adjacent selections.

Adding Console support

You can log to the console exactly as you would by calling console.log(), console.warn() or console.error(). The only constraint is that these methods only take a single argument.
